# SMC

Full-stack JavaScript social media application based on Node JS


## Tools and Technologies

 - Node JS
 - Express JS
 - EJS
 - Mongoose


## Features

- Follow/Unfollow Users
- Update Biography,Password,Photos
- Responsive mode for all screen
- Filter photos with tags
